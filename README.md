# Mandelbrot-set-
Mini proyecto en Python para generar Conjunto de Mandelbrot.

Este es un pequeño proyecto para crear un pequeño fractal. 
el fractal tiene las siguientes dimensciones.

columnas: 3000
Filas: 3000
